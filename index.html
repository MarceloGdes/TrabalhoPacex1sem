História

Paginas web no inicio da internet geralmente não tinham nada de interatividade, pois eram usadas somente o HTML para estruturar a pagina e o CSS para estilização. Então a Netscape, dona do navegador mais famoso do inicio dos anos 90, desenvolveu o LiveScript, que depois de um tempo teve seu nome alterado para JavaScript em razão de uma parceria com a Sun, empresa responsável pelo JAVA.

O JavaScript tinha como principal objetivo dar o dinamismo a pagina, como mapas, gráficos, animações, operações numéricas, formulários, infográficos interativos. Portanto se tornou um dos elementos principais no front-end do desenvolvimento web com uso ou não de frameworks (jQuery, ReactJS, Vue.JS e Angular).
